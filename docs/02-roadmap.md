# Roadmap

Version : 1.0

Statut : Validé

---

# Objectif

Cette roadmap décrit les grandes étapes de développement de JapaneseLearningApp.

Chaque sprint doit produire une application plus utile pour apprendre le japonais.

---

# Sprint 0 ✅

## Objectif

Préparer l'environnement de développement.

## Réalisé

- Installation de Git
- Configuration de Git
- Création du dépôt GitHub
- Installation d'Android Studio
- Création du projet Android
- Premier commit
- Mise en place de la documentation

Statut : **Terminé**

---

# Sprint 1

## Objectif

Préparer les fondations de l'application.

## À réaliser

- Ajouter les dépendances principales
  - Navigation Compose
  - Hilt
  - Room
- Organiser les packages du projet
- Créer la navigation
- Créer un écran d'accueil
- Vérifier que l'application démarre correctement

Résultat attendu :

Une application vide mais correctement structurée.

---

# Sprint 2

## Objectif

Afficher les hiragana.

## À réaliser

- Créer le modèle `Kana`
- Ajouter les données des hiragana au format JSON
- Lire les données
- Afficher la liste des hiragana
- Créer le premier écran d'apprentissage

Résultat attendu :

L'utilisateur peut consulter tous les hiragana.

---

# Sprint 3

## Objectif

Créer les flashcards.

## À réaliser

- Affichage d'une carte
- Navigation entre les cartes
- Lecture de l'audio
- Affichage du romaji

Résultat attendu :

L'utilisateur peut commencer à apprendre les hiragana.

---

# Sprint 4

## Objectif

Mettre en place la répétition espacée.

## À réaliser

- Historique des révisions
- Calcul des prochaines révisions
- Cartes à revoir

Résultat attendu :

L'application propose automatiquement les caractères à réviser.

---

# Sprint 5

## Objectif

Créer les quiz.

## À réaliser

- Questions de reconnaissance
- Vérification des réponses
- Enregistrement des résultats

Résultat attendu :

L'utilisateur peut tester ses connaissances.

---

# Sprint 6

## Objectif

Ajouter le suivi de progression.

## À réaliser

- Statistiques
- Progression globale
- Nombre de caractères maîtrisés

Résultat attendu :

L'utilisateur peut suivre ses progrès.

---

# Version 1

La Version 1 sera considérée comme terminée lorsque l'application permettra :

- d'apprendre les hiragana ;
- d'apprendre les katakana ;
- d'utiliser les flashcards ;
- d'écouter la prononciation ;
- de réaliser des quiz ;
- de bénéficier de la répétition espacée ;
- de suivre sa progression.

L'objectif principal reste de proposer une application simple, agréable et réellement utilisée au quotidien pour apprendre le japonais.
